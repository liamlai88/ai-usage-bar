## 四、交互细节

| 触发              | 反应                                       |
|-------------------|--------------------------------------------|
| 用量跨过阈值       | 小猫切换情绪，伴随一次"惊讶"动画           |
| 点击小猫           | 喵叫一声 (可选音效)，弹一个文字气泡显示详情 |
| 5h 重置剩 < 5min   | 头顶冒出 ⏰ 图标提醒                       |
| 周用量 > 90%       | 浮窗周围添加红色脉动光晕                   |
| 长时间无变化       | 自动进入"打盹"状态降低 CPU                  |

## 五、技术实现选择 (需要你选)

### 方案 A：纯 emoji 字符画 (最简单)
- 用 `tkinter` 透明窗口画 emoji 文本
- 每帧切字符串
- 优点：1 天能做完，无外部资源
- 缺点：表情有限，不够"游戏感"

### 方案 B：PyQt + 像素 sprite 图 (推荐)
- 用 `PyQt6` 画带 alpha 通道的 PNG sprite
- 每状态有 2-3 帧 PNG 资源
- 我用 emoji + PIL 在线生成 sprite，或者你提供素材
- 优点：质感好，可商用化
- 缺点：要装 PyQt，sprite 资源需要准备

### 方案 C：Übersicht 桌面 widget (HTML/CSS)
- 用 CSS 动画做小猫
- 可以做更骚的特效
- 优点：超灵活，社区资源多
- 缺点：要装 Übersicht，跟 Python widget 双跑

### 方案 D：保留菜单栏 + emoji 动画 (最轻)
- 不做浮窗，只让菜单栏图标在 `😺 😸 😻 😼 🙀 💥` 之间动起来
- 标题文字每 800ms 切一帧
- 优点：零新依赖
- 缺点：动得不够"猫"

## 六、参考的优秀 widget 设计

- **Tamagotchi** / NeoPets：经典电子宠物，情绪+喂养

- **Twitter 上 #pixelart 桌宠**：32×32 像素，Tamagotchi 美学
- **Cleanshot 顶栏**：极简数据 + 微动效
- **Reeder 5 的猫图标**：静态但精致

## 七、我推荐

**方案 D（菜单栏动画）+ 方案 B（浮窗小猫）双管齐下**：
1. 菜单栏先做表情动画（10 分钟工作量，立刻看到效果）
2. 浮窗用 PyQt 像素小猫，分两步：
   - Phase 1：用 emoji 字符做 placeholder 跑通逻辑
